# Prefect Docs / FastMCP Proxy

This server shows how to use FastMCP as a proxy for a remote MCP server.

It is configured to work with the Prefect documentation MCP server, so you can ask questions about Prefect’s docs.

💡 Consider using the [official Prefect MCP server](https://github.com/PrefectHQ/prefect-mcp-server) instead. It includes this functionality and can also interact with your Prefect servers!
